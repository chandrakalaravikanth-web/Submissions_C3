# Multimodal AI Design Analysis Suite
* What it demonstrates:
An app that lets users upload product/app designs.
* Create multiple agents like
** visual analysis agent
** UX critique agent
** market research agent.
** And more
* These agents reason over images, compare designs, and recommend improvements—all with structured, actionable output.
* Why it's ideal:
Leverages image RAG, Multimodal LLMs, AI Agents and design best practices. 
* Perfect for product, UI/UX, and dev teams—shows how GenAI can automate review and market intelligence.
